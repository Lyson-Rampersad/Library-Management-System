# Library Management System

## Overview

This project is a **Python-based Library Management System** developed as part of my **Higher Certificate in Information Technology**. It simulates the daily operations of a library by allowing staff to manage books, members, loans, returns, fines, and multiple library branches through a menu-driven console application.

The project was designed to apply core programming concepts to a realistic business scenario while demonstrating structured program design, modular development, and problem-solving skills.

---

## Features

* User authentication for library staff
* Book catalogue management
* Member registration and management
* Borrowing and returning books
* Fine calculation based on membership type
* Multiple library branch support
* Report generation
* Input validation and error handling

---

## Technologies Used

* Python 3
* Python Standard Library

---

## Project Structure

main.py      - Application entry point and menu system
members.py   - Member management functions
books.py     - Book catalogue management
loans.py     - Loan, return and fine processing
reports.py   - Reporting functionality
data.py      - Sample data and system storage
